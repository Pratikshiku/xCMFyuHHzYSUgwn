## House Rental Information System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0243dacc21894930904de2d9a3c9be56/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Property Information
5. Become a Landlord
6. Announcement Information
7. Message Feedback
8. Personal Center
9. Backend Management (Personal Center, Schedule Property Viewing, Contract Information, Property Maintenance Request, Maintenance Processing, Property Review, Become a Landlord and My Favorites Management)


**Regular Landlord Functions:**
1. Login
2. Homepage
3. Personal Center
4. Property Information Management
5. Property Viewing Management
6. Contract Information Management
7. Property Maintenance Management
8. Maintenance Processing Management
9. Property Review Management


**Administrator Functions:**
1. Login
2. Homepage
3. Personal Center
4. User Management
5. Property Type Management
6. Property Information Management
7. Property Viewing Management
8. Contract Information Management
9. Property Maintenance Management
10. Maintenance Processing Management
11. Property Review Management
12. Become a Landlord Management
13. Message Board Management
14. System Management
